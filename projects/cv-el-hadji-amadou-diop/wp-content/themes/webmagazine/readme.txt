Install:
install theme from the wordpress admin panel, or upload theme to server via ftp.
Wordpress admin panel install:
1) go to apperance -> themes
2) choose upload theme
3) find the zip file and click install

upload via ftp.

unzip folder and drag folder to ->public_html -> wp_content -> themes.

For support use the support forum at www.wpnewsmagazinethemes/forum.com
