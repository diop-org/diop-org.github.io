A) Requirements - top

The theme has been tested on all major browsers including Firefox 4,5 & 6, Opera 10 & 11, Internet Explorer 8 & 9 and Safari 5.1. Your browser should have JavaScript enabled to make this theme work! For example the DropDown Navigation and the Content Slider need Javascript to work.

The theme works on the latest Wordpress version without any problems. Previous versions have not been tested and are not supported. I suggest to update your wordpress installation, especially due to security flaws.

B) Theme Installation - top

  1. Automatic Installation

    Go to WP-Admin > Appearance > Themes > Install Themes > Upload and choose the theme zip folder.

  2. Manual Installation

    Upload the theme files to your wordpress theme folder wp-content/themes and activate the theme in your wordpress admin panel. That's it!

To find out more about installing WordPress themes please also see the WordPress documentation.

C) Theme Setup - top

Afterwards you should read a few more detailed explanations how to completely setup your downloaded theme.

  1. Pagination

    The theme includes css styles for the 'wp_pagenavi' plugin. Simply download the plugin here and install it to use pagination. Then deactivate the plugin css in the plugin options page to use the CSS of the theme.

  2. Contact Form

    The theme includes css styles for the 'Contact Form 7' plugin. Simply download the plugin here and build your contact form in the plugin options page. Then you can include your form easily in every page via a simple shortcode.

  3. Personal Picture

    The theme uses the wordpress custom header function to display your personal profile picture at the sidebar. You can upload your own picture by clicking on WP-Admin > Appearance > Header. If you upload more than one picture, you can choose the Random Mode, which shows a different image on each page.

    You can also choose an individual picture at every page by using the post_thumbnail function. Just edit a page, click on Set featured image, upload your picture and click Use as featured image

D) Theme Options - top

The theme contains an own admin page for further settings. Login to your Wordpress admin panel and go to WP-Admin > Appearance > Theme Options. There are several settings to change the logo, layout, colors and fonts of your theme. Furthermore you can manage the Featured Post Slider, your Personal Details and Networks. Every theme option is quite well explained so I hope there shouldn't be any questions.

These settings should definitely reduce the launch time for your wordpress business card. For any other changes which can't be made with the admin page you have to manually change the files.

E) Page Templates - top

With WordPress you can use different types of templates for a page. In order to choose your page template you have to create a new page or edit an existing page. You will find the Page Attributes area on your sidebar beside the editor screen. From that area you can choose your page template.

I have included a few page templates with this theme.

  1. Networks Template

    This template shows all network buttons like twitter, facebook etc.

  2. Slider Template

    Shows a static page with the featured posts slider included.

  3. Sitemap Template

    Shows a Sitemap with the latest posts, pages, categories and archives.

  4. Portfolio Templates (2 or 3 columns)

    The portfolio page templates shows the latest portfolio items. The Portfolio is managed with custom post types and has a own menu in WP-Admin to create and edit portfolio items.

F) Shortcodes - top

My card theme isn't bloated with a bunch of shortcodes and uses only a few, but important shortcodes to handle the business card features. You can insert all shortcodes easily by using the included buttons in the visual editor screen.

  1. Resume Shortcode

    Add Job Title, Organisation, date and a description and your ready. Using Shortcodes for the Resume is the most flexible and powerful option you can have.

  2. Skills Shortcode

    Add your skill name, your skill level from 1-5 stars and an optional description.

  3. Plugin Shortcode

    Here you can easily create a shortcode to include the theme card plugins like the 'Personal Details' Box and 'Network Buttons'
