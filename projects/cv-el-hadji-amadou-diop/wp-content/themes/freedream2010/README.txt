FreeDream 2010 Theme README.

The FreeDream 2010 theme for WordPress 3.0+ is stylish, customizable, and readable Ð make it yours with a custom menu, and background. (Headers IMG commented Out!)
2010 FreeDream is build on the default Twenty Ten Theme and supports six widgetized areas (two in the sidebar, four in the footer) and featured images (thumbnails for gallery posts).
The 2010 FreeDream are valid HTML5 and using CSS3 properties.

# Basic Setup of the Theme.
-----------------------------------
Fresh installation!

1. Upload the 2010 FreeDream Theme folder to your wp-content/themes folder.
2. Activate the theme from the WP Dashboard.
3. Done!

# Top Navigation Menu:
- Default the page-links start from the right! Use the Menus function in Dashboard/Appearance to rearrange the buttons and build your own Custom-menu. DO NOT USE LONG PAGE NAMES, Maximum 14 letters/numbers incl. spaces!
- Read more here: http://codex.wordpress.org/WordPress_Menu_User_Guide

# Twitter button:
- UNCOMMENT the lines in the sidebar.php, IF YOU WANT TO PROVIDE YOUR TWITTER LINK. Replace YOUR_TWITTER_NAME in the link with your own name.

# Add a Sitemap:
To activate the "Sitemap-page": add a New Page, name it "sitemap", choose "Sitemap" in the Template options and save it. That's all! (You can rename it after the page is published.)

# Header IMG:
- This function is default <!-- Commented Out --> in the FreeDream 2010 Theme!

# Localization:
- Copy/paste the languages-files from your local Twenty Ten Theme to FreeDream2010. 
- Danish, German and Spanish .po/.mo files is included.
