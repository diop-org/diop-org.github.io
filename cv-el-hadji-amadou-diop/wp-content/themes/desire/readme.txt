== ABOUT DESIRE THEME ==

* Use theme settings to customize this theme
* More updates will keep coming
* Instructions on how to add new items to your post slideshow is provided in the /documentation folder under theme's parent folder.
* Theme Installation steps: 1). Unzip the desire theme folder in /wp-content/themes folder 2) Go to Appearance -> Themes and activate the theme.
